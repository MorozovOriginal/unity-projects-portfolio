# Unity Projects Portfolio

Простой сайт-портфолио для проектов Unity.  
На главной странице есть карточки проектов с названием, описанием, основной картинкой или GIF и модальным окном с дополнительными изображениями.

## Что внутри

- `index.html` — структура сайта
- `style.css` — оформление
- `script.js` — список проектов и логика модального окна
- `assets/` — папка для картинок и GIF


## Как изменить блок «Обо мне»

В файле `index.html` найди блок:

```html
<aside class="profile-card">
```

Внутри можно заменить:

- `Имя Фамилия`
- возраст
- город
- email
- Telegram или Discord
- короткое описание себя

Например:

```html
<h2>Алексей Иванов</h2>
<strong>15 лет</strong>
<strong>Санкт-Петербург</strong>
<strong>alex@example.com</strong>
<strong>@alex_dev</strong>
```

## Как добавить проект

Открой файл `script.js` и найди массив `projects`.

Каждый проект выглядит так:

```js
{
  title: "Название проекта",
  description: "Краткое описание проекта.",
  cover: "assets/project-1-cover.gif",
  images: [
    "assets/project-1-screen-1.png",
    "assets/project-1-screen-2.png"
  ]
}
```

Чтобы добавить новый проект:

1. Скопируй один такой блок.
2. Вставь его ниже предыдущего проекта.
3. Измени название, описание и пути к картинкам.
4. Загрузи свои изображения в папку `assets`.

## Как удалить проект

В файле `script.js` удали весь блок проекта из массива `projects`.

Главное — следи, чтобы между проектами стояли запятые, кроме последнего проекта.

## Как заменить картинки

1. Загрузи свою картинку или GIF в папку `assets`.
2. Укажи путь к ней в `script.js`, например:

```js
cover: "assets/my-game-cover.gif"
```

Для дополнительных картинок:

```js
images: [
  "assets/my-game-1.png",
  "assets/my-game-2.png",
  "assets/my-game-3.png"
]
```

Картинок может быть любое количество: одна, две, три, шесть и больше.  
Сетка в модальном окне подстроится автоматически.
